---
title: "Camera"
description: "Camera-based features for boards with onboard sensors"
weight: 45
---

Camera features are available on builds that enable `CONFIG_HAS_CAMERA`. The camera module supports:

- **Live camera streaming** — MJPEG stream viewable in a browser with real-time quality, resolution, and FPS controls.
- **Motion detection** — On-device motion detection with optional SD snapshots and Discord webhook alerts.

The **Seeed Studio XIAO ESP32-S3 Sense** is the primary supported camera build.
