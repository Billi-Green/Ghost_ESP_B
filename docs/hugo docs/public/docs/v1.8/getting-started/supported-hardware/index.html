Supported Hardware

Boards and ESP32 variants officially supported by GhostESP

On this page

  • Display-enabled boards
  • Generic development boards
  • Notes

GhostESP runs on a range of ESP32 development boards and consumer devices. Confirm your hardware is listed here before flashing.

Display-enabled boards

  • Cheap Yellow Display (CYD) family
    • CYD2USB (USB-C)
    • CYDMicroUSB
    • CYDDualUSB (dual connectors)
    • AITRIP CYD (ESP32-2432S028R)
    • CYD 2.4″ variants
  • Large-format displays
    • Waveshare 7" LCD (ESP32-S3)
    • Crowtech 7" LCD (ESP32-S3)
    • Sunton 7" LCD (ESP32-S3)
  • Handheld displays
    • ESP32-S3 Cardputer / Cardputer ADV
    • Marauder V4 / V6 (240×320 touchscreen)
    • Awok Mini (128×128 display)
    • Awok ESP32 v5
    • LilyGo T-Watch S3
    • LilyGo T-Embed C1101
    • LilyGo T-Display S3 Touch
    • LilyGo T-Deck

Generic development boards

  • ESP32 (original dual-core)
  • ESP32-S2 (Wi-Fi only)
  • ESP32-S3
  • ESP32-C3
  • ESP32-C5
  • ESP32-C6

Notes

  • Some boards only expose IR transmit or lack SD card slots; check the board-specific guide before flashing.
  • Performance and available features depend on the ESP32 variant and peripherals.
  • Beta firmware builds may add experimental hardware—review the release notes for the latest additions.
